1
앱 시작하기

📱 홈 화면에 추가하면 앱처럼 사용할 수 있어요

스탬핏은 별도 설치 없이 브라우저에서 바로 쓸 수 있는 웹앱(PWA)입니다.
홈 화면에 추가하면 아이콘으로 실행할 수 있어요.

🤖 안드로이드 홈 화면 추가 방법 (Chrome 기준)
1
Chrome으로 스탬핏 접속
Chrome 주소창에 스탬핏 주소를 입력해 접속하세요.
2
Chrome 메뉴 열기
화면 우측 상단 점 세 개(⋮) 버튼을 탭합니다.
3
"홈 화면에 추가" 또는 "앱 설치" 선택
메뉴에서 "홈 화면에 추가" 또는 "앱 설치"를 탭합니다. (Chrome 버전에 따라 표현이 다를 수 있어요)
4
추가 확인
팝업에서 "추가"를 탭하면 홈 화면에 스탬핏 아이콘이 생성됩니다.
💡
홈 화면 아이콘을 탭하면 주소창 없이 앱처럼 실행됩니다. 데이터는 인터넷 없이도 유지됩니다.
🚀 처음 실행 — 빠른 시작

앱을 처음 열면 온보딩 화면이 나타납니다. 공연명만 입력하면 바로 시작할 수 있어요.

1
공연명 입력
입력창에 관람할 공연명을 입력하세요. (예: 뮤지컬 베르테르)
2
시작 방법 선택
직접 설정하기 — 좌석 등급, 할인 권종, 도장판을 직접 설정
빠른 시작 — 공연명만 입력하고 나중에 설정
ℹ️
재관람 할인 유저라면 "직접 설정하기"를 추천합니다.
좌석 등급과 할인 권종을 미리 설정해두면 일정 추가 시 금액이 자동 계산됩니다.
2
화면 구조 한눈에 보기

⠿

⚙

좌석 등급, 할인 권종, 배수 이벤트, 특별 이벤트, 데이터 관리
💡

3
공연 등록하기
🎭 새 공연 추가
1
+ 버튼 탭
화면 우측 상단 + 버튼을 탭합니다.
2
공연 정보 입력
공연명 (필수, 최대 30자)
공연장 (선택)
공연 기간 — 시작일 ~ 종료일 (선택)
대표 색상 — 6가지 중 선택
3
저장
하단 저장 버튼을 탭합니다.
💡
공연 기간을 입력해두면 공연 종료 후 자동으로 보관 제안 알림이 나타납니다.
✏️ 공연 수정 / 삭제 / 보관

상단 공연 탭을 길게 누르면 메뉴가 나타납니다.

수정 보관 탭 순서 변경 삭제
⚠️
공연 삭제는 관련된 모든 일정·도장판 데이터가 함께 삭제됩니다. 복구할 수 없으니 신중하게 선택하세요.
데이터를 보존하려면 보관을 사용하세요.
4
도장판 만들기
🟣 새 도장판 추가
1
현황 탭 → "+ 새 판" 탭
현황 탭 도장판 섹션 우측의 + 새 판 버튼을 탭합니다.
2
판 이름 입력
기본값으로 "1판", "2판" 등이 자동 입력됩니다. 원하는 이름으로 변경 가능합니다.
3
칸 수 설정
기본 7칸. 공연사 재관람 카드 칸 수에 맞게 조정하세요. (최소 1, 최대 50)
4
혜택 설정 (선택)
도장 N개 달성 시 받는 혜택을 입력합니다.
예: 5개 → 재관람 30% 할인쿠폰, 10개 → 무료 관람권
5
기존 도장 있으면 입력
앱 등록 전에 이미 찍혀있는 도장이 있다면 토글을 켜고 개수를 입력하세요.
ℹ️
두 번째 도장판부터는 이전 판의 칸 수와 혜택 설정이 자동으로 적용됩니다. "변경하기"를 탭하면 직접 수정할 수 있어요.
🎨 도장판 색상 변경

도장판마다 다른 색상을 지정할 수 있습니다. 도장판이 여러 개일 때 구분하기 좋아요.

판 추가 또는 편집 화면에서 12가지 색상 중 선택할 수 있습니다.

5
관람 일정 추가하기
📅 일정 추가 — 두 가지 방법

방법 A: FAB 버튼 탭 (상세 입력)

플래너 탭 우측 하단 보라색 + 버튼을 탭하면 일정 추가 화면이 열립니다.

방법 B: FAB 버튼 길게 누르기 (빠른 추가)

+ 버튼을 0.5초 이상 길게 누르면 마지막으로 사용한 좌석 등급·할인 권종이 자동 입력된 빠른 추가 화면이 열립니다.

💡
같은 공연을 반복 관람할 때는 길게 누르기를 활용하면 훨씬 빠르게 일정을 추가할 수 있어요.
✍️ 일정 입력 항목
①
언제 보시나요?
오늘 봤어요 — 오늘 날짜 자동 설정 후 바로 확정으로 진행
나중에 볼 예정 — 예정 날짜 직접 입력
②
날짜 / 회차 시간
관람 날짜를 선택합니다. 배수 이벤트 기간이면 자동으로 배수가 설정되고 안내 배너가 표시됩니다.
③
좌석 등급 / 할인 권종
드롭다운에서 선택하면 정가 → 할인가 → 절약 금액이 자동 계산됩니다.
④
나눔 관극 토글 (선택)
나눔 관극이라면 토글을 켜세요. 관람 기록은 남지만 도장은 적립되지 않습니다.
⑤
캐스트 메모 (선택)
캐스트를 입력하면 이후 통계에서 확인할 수 있습니다. 이전 입력값이 자동완성으로 제안됩니다.
⑥
특별 이벤트 태그 (선택)
싸인회, 하이터치회, 폴라로이드데이 등 이 회차의 특별 이벤트를 선택합니다.
6
관람 확정하기 (도장 찍기)
📌
언제 확정하나요?
재관람 할인권으로 티켓을 수령하는 시점에 앱을 열어 확정하는 것을 권장합니다. 확정해야 도장이 도장판에 기록됩니다.
✅ 오늘의 퀵카드로 빠르게 확정

오늘 날짜의 미확정 일정이 있으면 플래너 탭 최상단에 퀵카드가 나타납니다.

플래너 탭 최상단 오늘의 퀵카드 확인
"확정하기" 버튼 탭 → 관람 확정 화면 오픈
재관람표 제출 여부 체크 (재관람 권종일 때)
하단 "확정하기" 버튼 탭 → 도장 자동 적립 완료 🎉
🎟️ 재관람 권종 유저 주의사항
!
확인 사항 체크 필수
재관람 권종으로 예매한 일정은 확정 화면에서 "재관람표를 제출했어요"를 반드시 체크해야 확정 버튼이 활성화됩니다.
💡
체크리스트는 재관람표를 제출하기 전에 확인용으로 사용하세요. 공연장 입장 전, 티켓 수령 시점에 앱을 여는 것을 권장합니다.
↩️ 확정 취소

확정 후 24시간 이내에만 취소할 수 있습니다.

1
일정 카드 ··· 메뉴 탭
확정된 일정 카드 우측 상단 ··· 버튼을 탭합니다.
2
"확정 취소" 선택
24시간이 지났다면 메뉴에서 보이지 않습니다.
⚠️
확정 취소 시 해당 일정으로 적립된 도장이 회수됩니다.
🌀 배수 이벤트 (더블·트리플 적립)

특정 기간 동안 도장을 2~5배로 적립하는 이벤트를 등록할 수 있습니다.

이벤트 기간의 날짜로 일정을 추가하면 배수가 자동으로 설정되고 안내 배너가 표시됩니다.

💡
배수 이벤트 등록은 설정 탭 → 배수 이벤트에서 할 수 있어요.
7
현황 탭 보는 법
📊 현황 탭 구성 (위→아래)
①
요약 카드
총 관람 횟수 / 다음 혜택까지 남은 도장 수 / 확정 일정 수 / 미사용 혜택 수
②
💡 FocusCard (추천 안내)
"1판에 2개만 더 찍으면 할인쿠폰 30%이에요" 형태로 가장 가까운 혜택을 강조합니다.
③
도장판 카드
도장판별 진행 현황(채워진 칸 / 전체 칸)과 다음 혜택까지 남은 개수가 표시됩니다.
④
혜택 현황
달성한 혜택과 미달성 혜택 목록. 달성한 혜택은 amber(노란) 배경으로 강조됩니다.
⑤
더보기 (기본 접힘)
"더보기"를 탭하면 관람 비용 요약, 캐스트 통계, 배분 시뮬레이터가 나타납니다.
🎯 배분 시뮬레이터 사용법

앞으로 N번 더 관람할 예정일 때 도장을 어떻게 배분하면 혜택을 가장 많이 받을 수 있는지 미리 확인할 수 있습니다.

1
현황 탭 → 더보기 → 배분 시뮬레이터
"배분 시뮬레이터" 버튼을 탭합니다.
2
남은 관람 횟수 입력
앞으로 몇 번 더 볼 예정인지 입력합니다.
3
결과 확인
도장판별로 몇 개가 추가되고 어떤 혜택을 달성할 수 있는지 보여줍니다.
ℹ️
시뮬레이터는 실제 데이터를 변경하지 않습니다. 순수하게 예측만 합니다.
💰 금액 숨기기

현황 탭 비용 섹션 우측 눈 아이콘(👁)을 탭하면 금액을 블러 처리할 수 있습니다.

스크린샷 촬영이나 화면 공유 시 금액이 노출되는 것을 방지할 수 있어요.

8
혜택 사용 처리하기
⭐ 달성 혜택 사용 완료 처리

도장을 N개 채워 혜택을 달성하면 현황 탭 혜택 현황에서 사용 버튼이 나타납니다.

1
현황 탭 → 혜택 현황 확인
amber(노란) 배경의 달성된 혜택을 찾습니다.
2
"사용" 버튼 탭
확인 팝업이 나타납니다. "확인"을 탭하면 사용 완료 처리됩니다.
3
사용 날짜 자동 기록
사용 완료된 혜택은 회색 + 취소선으로 표시되고 날짜가 기록됩니다.
💡
실수로 사용 처리했다면 "취소" 버튼으로 즉시 되돌릴 수 있어요.
🎫 할인쿠폰 트래커

혜택 이름에 "할인쿠폰" 또는 "할쿠"가 포함된 달성 혜택은 상단에 트래커로 자동 집계됩니다.

예: 🎫 할인쿠폰 1 / 3개 사용

ℹ️
대본집, 폴라로이드, OST 등 수집형 혜택은 "사용" 버튼이 표시되지 않습니다. 해당 키워드가 혜택 이름에 포함되면 자동으로 수집형으로 분류됩니다.
9
설정 탭 활용하기
🎫 좌석 등급 설정

등급명과 정가를 등록합니다. 일정 추가 시 드롭다운에서 선택하면 할인가가 자동 계산됩니다.

예: VIP – 130,000원 / R – 110,000원 / S – 90,000원

🏷️ 할인 권종 설정

재관람 할인, 마티네 할인 등 할인 종류를 등록합니다.

①
할인 방식 선택
정률(%) — 정가의 N% 할인
정액(원) — N원 할인
직접 입력 — 최종 금액 직접 입력
②
재관람 여부 / 쿠폰 여부 체크
재관람: 체크하면 확정 시 "재관람표 제출" 체크리스트가 표시됩니다.
쿠폰: 체크하면 확정 시 "쿠폰 제출" 체크리스트가 표시됩니다.
✨ 특별 이벤트 관리

싸인회, 하이터치회, 럭키드로우 등 회차별 특별 이벤트 목록을 관리합니다.

기본 제공 프리셋 외에 직접 추가·수정·삭제할 수 있습니다. (이름 최대 20자)

⚡ 배수 이벤트 등록

특정 기간 도장을 2~5배로 적립하는 이벤트를 등록합니다.

1
이벤트명 입력
예: 더블 위크, 오픈런 이벤트
2
기간 설정
시작일 (필수) / 종료일 (선택, 없으면 무기한)
3
배수 선택
x2 (더블) / x3 (트리플) / x4 / x5
💾 데이터 관리
📤
JSON으로 내보내기
전체 데이터를 JSON 파일로 다운로드합니다. 기기 변경이나 백업에 활용하세요.
파일명: stampit-backup-YYYY-MM-DD.json
📥
JSON에서 가져오기
이전에 내보낸 JSON 파일을 불러옵니다. 기존 데이터가 모두 교체됩니다.

자주 묻는 질문
미확정과 확정의 차이가 뭔가요?
미확정은 예정된 일정만 기록된 상태입니다. 실제로 관람하고 재관람표 등을 제출한 후 확정해야 도장이 도장판에 기록됩니다.
재관람 할인권을 사용했는데 도장판에 반영이 안 돼요.
일정을 추가한 후 반드시 확정까지 완료해야 합니다. 플래너 탭의 미확정 일정 카드 하단 "확정하기" 버튼을 탭하세요.
공연이 여러 개인데 탭이 너무 많아졌어요.
공연 탭을 길게 누르면 "보관" 메뉴가 나타납니다. 보관하면 탭에서 숨겨지고 데이터는 유지됩니다. 설정 탭에서 공연 탭 순서도 변경할 수 있어요.
도장을 실수로 더 많이 찍었어요.
확정 후 24시간 이내라면 일정 카드 ··· 메뉴에서 "확정 취소"를 탭할 수 있습니다. 24시간이 지났다면 현황 탭 도장판의 ··· → "배분 조정"에서 수동으로 수정할 수 있어요.
폰을 바꿔도 데이터가 유지되나요?
스탬핏은 기기에만 데이터를 저장합니다. 기기 변경 전에 반드시 설정 탭 → JSON으로 내보내기로 백업하고, 새 기기에서 가져오기를 하세요.
나눔 관극은 어떻게 기록하나요?
일정 추가 시 "나눔 관극" 토글을 켜세요. 관람 기록은 남지만 도장판에 도장이 찍히지 않습니다. 일정 카드에 "나눔" 뱃지가 표시됩니다.
특별 이벤트 필터는 어떻게 사용하나요?
플래너 탭 상단 이벤트 필터 칩에서 원하는 이벤트를 탭하면 해당 이벤트가 태그된 일정만 표시됩니다.
빠른 계산기는 뭔가요?
일정 추가 없이 좌석 등급과 할인 권종을 선택해 할인가를 바로 확인할 수 있는 도구입니다. 플래너 탭 상단에서 "빠른 계산기"를 탭해 펼칠 수 있어요.
